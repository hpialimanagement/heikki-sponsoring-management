# Sponsoring Management System

Eine vollständige Sponsoring-Verwaltungssoftware mit Backend und Datenbank.

## Features

- ✅ Passwortschutz (Management)
- ✅ Sponsoren-Verwaltung (Add, Edit, Delete)
- ✅ 6 Status-Werte
- ✅ 5 Sponsoring-Kategorien
- ✅ Dashboard mit Statistiken
- ✅ Filter & Suchfunktion
- ✅ Notizen-Feld
- ✅ Datum-Tracking

## Installation

```bash
npm install
npm start
```

Die Website läuft dann auf `http://localhost:3000`

## Deployment auf Railway/Heroku

1. Repository auf GitHub pushen
2. Railway/Heroku mit GitHub verbinden
3. Automatisches Deployment

## Datenbank

Die Datenbank wird als `sponsors.json` gespeichert und ist Teil des Repositories.
